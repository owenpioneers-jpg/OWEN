About Me

My journey into web development started with Rab Web Solutions, my very first website. It wasn’t perfect. Layouts broke, JavaScript failed, and responsiveness tested my patience. But that project taught me something important: real skill is built by solving real problems.

From there, I kept building. Each project pushed my technical understanding further, from structuring clean HTML and responsive CSS to writing functional JavaScript and designing user-focused UI/UX.

Projects like Rab Cap, Rab Watch Dealers, City Scope Pharmacy, Rab Fitness Gym, and Crop Sensei weren’t just websites—they were learning grounds. I learned to think like a user, optimize performance, and turn ideas into working digital products.

Today, I build responsive websites from scratch, focusing on clean code, performance, accessibility, user experience, and solutions that serve business goals. I bring consistency, discipline, and persistence to every project.
